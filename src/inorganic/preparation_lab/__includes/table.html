\(\ce{Fe (s) + H2SO4 (aq) -> FeSO4 (aq) + H2 (g)}\)
某些金属 + 强碱 \(\ce{Zn (s) + 2NaOH (aq) -> Na2ZnO2 (aq) + H2 (g)}\)
\(\ce{2Al (s) + 2NaOH (aq) + 2H2O (l) -> 2NaAlO2 (aq) + 3H2 (g)}\)
氢氧化铝
上/260		 铝盐溶液 + 氨水 \(\ce{Al2(SO4)3 (aq) + 6NH3 · H2O (aq) -> 2Al(OH)3 (s) + 3(NH4)2SO4(aq)}\)

上/277		 石灰石 + 稀盐酸 \(\ce{CaCO3 (s) + 2HCl (aq) -> CaCl2 (aq) + H2O (l) + CO2 (g)}\)
氨气
上/300		 加热亚硝酸钠 + 氯化铵溶液
排水集气法		 \(\ce{NaNO2 (aq) + NH4Cl (aq) -> NaCl (aq) + N2 (g) + 2H2O (l)}\)
一氧化氮
上/302		 铜 + 稀硝酸 (还原) \(\ce{3Cu(s) + 8HNO3(aq, \text{稀}) -> 3Cu(NO3)2 (aq) + 2NO (g) + 4H2O (l)}\)
二氧化氮
上/302		 铜 + 浓硝酸 \(\ce{Cu (s) + 4HNO3 (aq, \text{浓}) -> Cu(NO3)2 (aq) + 2NO2 (g) + 2H2O (l)}\)
燃烧 硝酸铅 \(\ce{2Pb(NO3)2 (s) -> 2PbO (s) + 4NO2 (g) + O2 (g)}\)
一氧化二氮（笑气）
上/303		 加热 硝酸铵 \(\ce{NH4NO3 (s) -> N2O (g) + 2H2O (g)}\)
硝酸
上/307		 加热 热硝酸钠晶体 \(\ce{NaNO3 (s) + H2SO4 (aq, \text{浓}) -> NaHSO4 (s) + HNO3 (g)}\)
氨气
上/321		 铵盐 + 碱 \(\ce{NH4NO3 (aq) + NaOH (aq) -> NaNO3 (aq) + H2O (l) + NH3 (g)}\)

少量二氧化锰 分解过氧化氢 \(\ce{2H2O2 (aq) -> 2H2O (l) + O2 (g)}\)

上/347		 过氧化钡 + 稀硫酸 \(\ce{BaO2 (s) + H2SO4 (aq) -> BaSO4 (s) + H2O2 (aq)}\)
硫化氢

硫化铁 + 稀硫酸 \(\ce{FeS (s) + H2SO4 (aq) -> FeSO4 (aq) + H2S (g)}\)
氯气
上/389		 燃烧 浓盐酸 + 二氧化锰
浓盐酸 + 高锰酸钾 \(\ce{2KMnO4 (aq) + 15HCl (aq, \text{浓}) -> 2KCl (aq) + 2MnCl2 (aq) + 8H2O (l) + 5Cl2 (g)}\)
盐酸
上/395		 加热 浓硫酸 + 氯化钠固体 不加热/稍微加热：
\(\ce{NaCl (s) + H2SO4 (aq,\text{浓}) -> NaHSO4 (aq) + HCl (g)}\)
再加热（500~600°C）：
\(\ce{NaHSO4 (aq) + NaCl (s) -> Na2SO4 (aq) + HCl (g)}\)
总反应式：
\(\ce{2NaCl (s) + H2SO4 (aq,\text{浓}) -> Na2SO4 (aq) + 2HCl (g)}\)
