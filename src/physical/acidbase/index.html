电离度

\(\alpha = \frac{\text{已电离分子数}}{\text{分子总数}} \times 100\% \)

平衡常数

对于 \(\ce{aA + bB <=> cC + dD}\),
\(K_\text{c} = \dfrac{[\text C]^c [\text D]^d}{[\text A]^a [\text B]^b}\)

电离常数 / 电离平衡常数

\(K_\text{a}\) 弱酸 \(K_\text{b}\) 弱碱 \[K_\text{a} = \dfrac{[\text H^+][\text A^-]}{[\text{HA}]}, K_\text{b} = \dfrac{[\text A^+][\text {OH}^-]}{[\text{AOH}]}\]

压强平衡常数

\[K_\text{p} = \dfrac{p_C^c \times p_D^d}{p_A^a \times p_B^b}\]

溶度积

对于 \(\ce{A_m B_n (s) <=> m A^{n+} + n B^{m-}}\) \[K_\text{sp} = \ce{[A^{n+}]^m [B^{m-}]^n}\]

水的离子积常数

\[K_\text{w} = \ce{[H+][OH-]} = 1 \times 10^{-14}\]
