阳离子 试剂 观察 方程式
铵离子 \(\ce{NH4+}\) \(\ce{NaOH}\)溶液 产生的气体（\(\ce{NH3}\)）可使湿润的红色石蕊试纸变蓝 \(\ce{NH4+ + OH- -> NH3 ^ + H2O}\)
钾离子 \(\ce{K+}\) 焰色反应 火焰呈浅紫色
钠离子 \(\ce{Na+}\) 焰色反应 火焰呈黄色
镁离子 \(\ce{Mg^{2+}}\) 过量\(\ce{NaOH}\)溶液 生成白色\(\ce{Mg(OH)2}\)沉淀，该沉淀能溶于\(\ce{NH4Cl}\)溶液 \(\ce{Mg^{2+} +2 OH- -> Mg(OH)2 v}\)
钙离子 \(\ce{Ca^{2+}}\) \(\ce{NaOH}\)溶液
\(\ce{NH3}\) 溶液		 与\(\ce{NaOH}\)反应，会产生白色沉淀
与\(\ce{NH3}\)反应不会产生沉淀物		 \(\ce{Ca^{2+} + 2 OH- -> Ca(OH)2 v}\)
铝离子 \(\ce{Al^{3+}}\) 适量\(\ce{NaOH}\)溶液 生成白色氢氧化铝絮状沉淀，
且该沉淀可溶于过量的\(\ce{NaOH}\)溶液		 \(\ce{Al^{3+} + 3 OH- -> Al(OH)3 v}\)
钡离子 \(\ce{Ba^{2+}}\) 稀盐酸 \(\ce{HCl}\)
硫酸钠 \(\ce{Na2SO4}\)		 先于稀盐酸反应，无明显现象产生，可排除是\(\ce{Ag+}\)
后再加硫酸钠，能生成白色沉淀		 \(\ce{Ba2+ + 2Cl- -> BaCl2}\)
\(\ce{BaCl2 + Na2SO4 -> BaSO4 + 2NaCl}\)
铁(II)离子 \(\ce{Fe^{2+}}\) 少量\(\ce{NaOH}\)溶液 先生成白色氢氧化亚铁沉淀
过后迅速变成灰绿色，
最后变红褐色氢氧化铁沉淀		 \(\ce{Fe2+ + 2OH- -> Fe(OH)2 v}\)
铁(III)离子 \(\ce{Fe^{3+}}\) \(\ce{NaOH}\)溶液 生成红褐色\(\ce{Fe(OH)3}\)沉淀 \(\ce{Fe3+ + 3OH- -> Fe(OH)3 v}\)
铜(II)离子 \(\ce{Cu^{2+}}\) \(\ce{NaOH}\)溶液 生成蓝色的\(\ce{Cu(OH)2}\)沉淀
加热后可转变为黑色固体的\(\ce{CuO}\)或红色\(\ce{Cu2O}\)或二者皆有		 \(\ce{Cu^{2+} + 2OH- -> Cu(OH)2 v}\)
